# Project Goals — angelnereira.com

## Objetivo Principal

Transformar el portafolio de un CV estático a una plataforma activa de gestión de carrera
que demuestre en tiempo real las capacidades técnicas del owner.

## Objetivos Específicos (prioridad descendente)

1. Demostrar impacto medible: métricas reales > títulos académicos
2. Ser 100% bilingüe (ES/EN) sin fricción para visitantes internacionales
3. Facilitar la descarga del CV de forma segura y profesional
4. Automatizar la gestión del contenido desde el admin panel
5. Optimizar para SEO: aparecer en búsquedas relevantes en Panamá y mercado internacional
6. Servir como hub de "learning in public" para construir audiencia técnica

## KPIs de Éxito

- Lighthouse Performance ≥ 90 en todas las páginas
- Lighthouse SEO ≥ 95 en todas las páginas
- 0 textos hardcodeados (cobertura i18n 100%)
- CV descargable en < 3 segundos
- Todas las secciones del admin utilizables en mobile
- Al menos 1 artículo de blog publicado

## Anti-objetivos (lo que NO es este proyecto)

- No es una plataforma multi-usuario
- No necesita internacionalización a más de 2 idiomas (ES + EN)
- No necesita un sistema de comentarios en el blog (al menos en MVP)
- No es un e-commerce
